Theory: Orthogonal Trajectory

What are Orthogonal Trajectories?

Orthogonal trajectories are families of curves that intersect another family of curves at right angles (90°). In mathematics, particularly in differential equations, finding orthogonal trajectories is an important application.

Mathematical Definition

Given a one-parameter family of curves F(x, y, c) = 0, an orthogonal trajectory is a curve that intersects each curve of the family at a right angle.

Method to Find Orthogonal Trajectories

To find the orthogonal trajectories of a given family of curves:

  1. Find the differential equation of the given family by eliminating the parameter
  2. Replace dy/dx with -dx/dy (or -1/(dy/dx)) to get the differential equation of orthogonal trajectories
  3. Solve this new differential equation to obtain the family of orthogonal trajectories

Example: Orthogonal Trajectories of Straight Lines

Problem:

Find the orthogonal trajectories of the family of straight lines through the origin: y = mx

Solution:

1. Given family: y = mx

2. Differentiate: dy/dx = m

3. But from y = mx, m = y/x

4. So differential equation is: dy/dx = y/x

5. For orthogonal trajectories, replace dy/dx with -dx/dy

6. So: -dx/dy = y/x ⇒ x dx + y dy = 0

7. Integrate: x²/2 + y²/2 = c/2 ⇒ x² + y² = c

8. Orthogonal trajectories are circles centered at origin

Applications of Orthogonal Trajectories

  • Electrostatics: Electric field lines and equipotential surfaces are orthogonal
  • Fluid Dynamics: Streamlines and equipotential lines in ideal fluid flow
  • Heat Transfer: Isotherms and heat flow lines
  • Cartography: Lines of longitude and latitude on Earth

Pretest Questions

Pre-experiment Assessment

Answer the following questions to test your preliminary knowledge about orthogonal trajectories.

Section A: Polynomial Functions (3 Questions)

Question 1:

Find the orthogonal trajectories of the family of parabolas y = ax².

Solution:

1. Given: y = ax² ⇒ a = y/x²

2. Differentiate: dy/dx = 2ax = 2(y/x²)x = 2y/x

3. Replace dy/dx with -dx/dy: -dx/dy = 2y/x

4. Rearranging: x dx + 2y dy = 0

5. Integrate: x²/2 + y² = c

6. Orthogonal trajectories: x²/2 + y² = c (ellipses)

Question 2:

Find orthogonal trajectories of the family of curves y = cx³.

Solution:

1. y = cx³ ⇒ c = y/x³

2. dy/dx = 3cx² = 3(y/x³)x² = 3y/x

3. For orthogonal trajectories: -dx/dy = 3y/x

4. So: x dx + 3y dy = 0

5. Integrate: x²/2 + (3/2)y² = c ⇒ x² + 3y² = 2c

Question 3:

Find orthogonal trajectories of the family: y² = 4ax (parabolas opening right).

Solution:

1. y² = 4ax ⇒ a = y²/(4x)

2. Differentiate: 2y(dy/dx) = 4a ⇒ dy/dx = 2a/y = 2(y²/(4x))/y = y/(2x)

3. For orthogonal trajectories: -dx/dy = y/(2x)

4. So: 2x dx + y dy = 0

5. Integrate: x² + y²/2 = c ⇒ 2x² + y² = 2c

Section B: Trigonometric Functions (2 Questions)

Question 4:

Find orthogonal trajectories of the family: y = c sin x.

Solution:

1. y = c sin x ⇒ c = y/sin x

2. dy/dx = c cos x = (y/sin x) cos x = y cot x

3. For orthogonal trajectories: -dx/dy = y cot x

4. So: -dx = y cot x dy ⇒ -tan x dx = y dy

5. Integrate: ln|cos x| = y²/2 + c'

Question 5:

Find orthogonal trajectories of the family: x = c cos y.

Solution:

1. x = c cos y ⇒ c = x/cos y

2. Differentiate: 1 = -c sin y (dy/dx) ⇒ dy/dx = -1/(c sin y) = -cos y/(x sin y) = -cot y/x

3. For orthogonal trajectories: -dx/dy = -cot y/x ⇒ dx/dy = cot y/x

4. So: x dx = cot y dy

5. Integrate: x²/2 = ln|sin y| + c

Section C: Exponential & Logarithmic Functions (3 Questions)

Question 6:

Find orthogonal trajectories of the family: y = ce^x.

Solution:

1. y = ce^x ⇒ c = ye^{-x}

2. dy/dx = ce^x = (ye^{-x})e^x = y

3. For orthogonal trajectories: -dx/dy = y

4. So: -dx = y dy

5. Integrate: -x = y²/2 + c ⇒ y² = -2x + 2c

Question 7:

Find orthogonal trajectories of the family: y = ln(cx).

Solution:

1. y = ln(cx) = ln c + ln x

2. dy/dx = 1/x

3. For orthogonal trajectories: -dx/dy = 1/x

4. So: -x dx = dy

5. Integrate: -x²/2 = y + c ⇒ y = -x²/2 - c

Question 8:

Find orthogonal trajectories of the family: e^x sin y = c.

Solution:

1. e^x sin y = c

2. Differentiate: e^x sin y + e^x cos y (dy/dx) = 0

3. So: dy/dx = -sin y/cos y = -tan y

4. For orthogonal trajectories: -dx/dy = -tan y ⇒ dx/dy = tan y

5. So: dx = tan y dy

6. Integrate: x = ln|sec y| + c'

Section D: Circles (2 Questions)

Question 9:

Find orthogonal trajectories of the family of circles: x² + y² = a².

Solution:

1. x² + y² = a²

2. Differentiate: 2x + 2y(dy/dx) = 0 ⇒ dy/dx = -x/y

3. For orthogonal trajectories: -dx/dy = -x/y ⇒ dx/dy = x/y

4. So: dx/x = dy/y

5. Integrate: ln|x| = ln|y| + ln|c| ⇒ y = cx (straight lines through origin)

Question 10:

Find orthogonal trajectories of the family of circles: (x - h)² + y² = h².

Solution:

1. (x - h)² + y² = h² ⇒ x² - 2hx + h² + y² = h² ⇒ x² + y² = 2hx

2. Differentiate: 2x + 2y(dy/dx) = 2h ⇒ h = x + y(dy/dx)

3. From original equation: h = (x² + y²)/(2x)

4. Equate: x + y(dy/dx) = (x² + y²)/(2x)

5. Solve for dy/dx, then replace with -dx/dy for orthogonal trajectories

6. Final result: y² = -x² + 2kx (orthogonal trajectories are also circles)
