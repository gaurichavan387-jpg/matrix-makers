Orthogonal Trajectory Solution

Original Equation & Solution

Original Equation:
y = x²
Orthogonal Trajectory Equation:
y = -½ ln|x| + C

Step-by-Step Solution

Step 1: Differentiate the original equation

Given: y = x²

Differentiate with respect to x: dy/dx = 2x

Step 2: Find slope for orthogonal trajectory

For orthogonal trajectories, the slope is the negative reciprocal:

dy/dx = -1/(2x)

Step 3: Solve the differential equation

Solve: dy/dx = -1/(2x)

Separate variables: dy = -1/(2x) dx

Integrate both sides: ∫ dy = ∫ -1/(2x) dx

Step 4: Perform integration

Left side: ∫ dy = y

Right side: ∫ -1/(2x) dx = -½ ∫ 1/x dx = -½ ln|x| + C

Combine: y = -½ ln|x| + C

Step 5: Final orthogonal trajectory family

The family of orthogonal trajectories is:

y = -½ ln|x| + C

where C is an arbitrary constant representing different curves in the family.
