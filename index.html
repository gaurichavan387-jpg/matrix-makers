y = x^2 + 3x - 5

2) Solve Given Equation

Solution Steps:

For the equation: y = x² + 3x - 5

1. To find x-intercepts, set y = 0: x² + 3x - 5 = 0

2. Using quadratic formula: x = [-3 ± √(3² - 4*1*(-5))] / (2*1)

3. x = [-3 ± √(9 + 20)] / 2 = [-3 ± √29] / 2

4. Solutions: x₁ = (-3 + √29)/2 ≈ 1.193, x₂ = (-3 - √29)/2 ≈ -4.193

5. Y-intercept: x = 0 → y = -5

6. Vertex: x = -b/(2a) = -3/2 = -1.5, y = (-1.5)² + 3*(-1.5) - 5 = -7.25

3) Compute Orthogonal Trajectory

Orthogonal Trajectory Calculation:

For the original equation: y = x² + 3x - 5

1. Differentiate: dy/dx = 2x + 3

2. Slope of orthogonal trajectory: m⊥ = -1/(dy/dx) = -1/(2x + 3)

3. Differential equation for orthogonal family: dy/dx = -1/(2x + 3)

4. Separate variables: dy = -dx/(2x + 3)

5. Integrate both sides: ∫dy = -∫dx/(2x + 3)

6. Solution: y = -(1/2) ln|2x + 3| + C

7. Orthogonal trajectory equation: y = -½ ln|2x + 3| + C

4) Show Final Solution

Summary of Results:

Type: Quadratic Equation (Parabola)
